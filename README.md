
这是一个多人独立练习项目。每个人围绕“个性化健身训练 Agent”主题，独立实现自己的版本。

## 项目目标

构建一个能够根据用户目标、训练时间、器械条件、身体状态和训练记录，生成训练建议并进行动态调整的健身 Agent。

## 基础功能

1. 用户档案
2. 动作库
3. 今日训练计划
4. 训练记录
5. 渐进调整
6. 周报总结

## 推荐规则

- 每个人只修改自己的 members 目录
- 公共要求放在 common 目录
- 不提交 API Key、密码、个人隐私数据
- 每次更新后在自己的 progress.md 记录进展
